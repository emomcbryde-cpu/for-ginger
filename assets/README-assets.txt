DROP YOUR FILES HERE
====================

This /assets folder is where your personal media goes. The website works
without any of these files (it degrades gracefully), but adding them makes
it complete.

1) MUSIC  ->  assets/music.mp3
   - Any soft, romantic instrumental works well.
   - The file MUST be named exactly:  music.mp3
   - Music never plays on its own. It starts on the first tap (mobile
     browsers block autoplay), and there's a small toggle button top-right.

2) PHOTOS ->  assets/photo1.jpg
              assets/photo2.jpg
              assets/photo3.jpg
   - Named exactly photo1.jpg, photo2.jpg, photo3.jpg (lowercase).
   - Any size; they display best around 1200x900 (landscape 4:3).
   - If a photo is missing, that slot simply disappears. If none exist,
     the whole "Favorite Memories" section stays hidden.

That's it — no other steps needed.
